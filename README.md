# I. Usage specifications

1. The ownership of this logo belongs to Jiangxi Normal University Association of Computer Science.
2. This logo prohibits commercial use, and it is prohibited to sell it through any means and ways.
3. When hosting an event at the college level or above, please ensure that the college logo is also used.
4. There are two sizes of icons, so if you are using the body of the logo on its own, please make sure that the English font used for "Assignment of Computer Science" is "Inter"(Open source, OFL-1.1 license, repo: [rsms/inter](https://github.com/rsms/inter)).
5. ...Other rules are not available now...
